# Sellio Bot — Инструкция по запуску

## Структура проекта

```
sellio/
├── bot.py                  # Точка входа
├── config.py               # Конфигурация
├── requirements.txt        # Зависимости
├── handlers/
│   ├── start.py            # /start, меню, статус
│   ├── analyze.py          # Анализ карточек, отзывы, история
│   ├── payment.py          # Тарифы, оплата, возврат
│   ├── referral.py         # Реферальная программа
│   └── admin.py            # Админ-панель
└── services/
    ├── database.py         # SQLite база данных
    ├── parser.py           # Парсер WB, Ozon, Яндекс Маркет
    ├── analyzer.py         # AI генерация через Claude API
    ├── payment_service.py  # ЮKassa интеграция
    └── scheduler.py        # Фоновые задачи
```

## Переменные окружения (Railway → Variables)

| Переменная | Где взять | Пример |
|---|---|---|
| `BOT_TOKEN` | @BotFather в Telegram | `7234567890:AAF...` |
| `ANTHROPIC_API_KEY` | console.anthropic.com | `sk-ant-...` |
| `YUKASSA_SHOP_ID` | yookassa.ru → Настройки | `123456` |
| `YUKASSA_SECRET_KEY` | yookassa.ru → Настройки | `test_...` |
| `ADMIN_IDS` | Твой Telegram ID (узнать у @userinfobot) | `123456789` |
| `DB_PATH` | Оставь по умолчанию | `sellio.db` |

## Деплой на Railway

### Шаг 1 — GitHub
1. Создай репозиторий на github.com (назови `sellio-bot`)
2. Загрузи все файлы проекта

### Шаг 2 — Railway
1. Зайди на railway.app → New Project → Deploy from GitHub
2. Выбери свой репозиторий
3. Railway автоматически обнаружит Python проект

### Шаг 3 — Переменные
1. В Railway → твой проект → Variables
2. Добавь все переменные из таблицы выше
3. Нажми Deploy

### Шаг 4 — Проверка
Напиши своему боту `/start` — должно прийти приветствие.

## Команды бота

**Для пользователей:**
- `/start` — главное меню
- `/status` — статус подписки
- `/refund` — запросить возврат
- `/menu` — открыть меню

**Для администратора:**
- `/admin` — статистика (пользователи, выручка, запросы)
- `/broadcast текст` — рассылка всем пользователям

## Тестирование оплаты

В ЮKassa есть тестовый режим — используй тестовые карты:
- Успешная оплата: `5555 5555 5555 4444`, любой CVV, любая дата
- Отклонённая: `5555 5555 5555 4477`

## Важно перед запуском

1. В ЮKassa → Интеграция → HTTP-уведомления — укажи URL вебхука
   (необязательно для старта, статус проверяется вручную кнопкой «Я оплатил»)
2. Убедись что `YUKASSA_SHOP_ID` и `YUKASSA_SECRET_KEY` от боевого аккаунта
   (не тестового) перед приёмом реальных платежей
3. Зарегистрируйся как ИП или самозанятый до получения первых платежей
